Como Funciona?

Este sistema utiliza IA (Inteligência Artificial) treinada previamente através do Teachable Machine. Ele analisa os quadros de vídeo capturados pela sua câmera e tenta classificar o que está vendo com base nas categorias que você ensinou.

Dica: Garanta uma boa iluminação para que a API (Application Programming Interface) do modelo consiga extrair as características com maior precisão.
